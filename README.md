# Netflix Movie Data Analysis

## Project Overview
This project analyzes Netflix movie data using Python. The goal is to explore patterns in movie ratings, genres, votes, and release years using data analysis techniques.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains information about Netflix movies such as:
- Movie Title
- Genre
- Release Year
- Rating
- Vote Count

## Analysis Performed
- Movie with the highest votes
- Distribution of movie ratings
- Genre analysis
- Release year trends

## Files in Repository
- `netflix_data_analysis.ipynb` – Jupyter Notebook containing the full analysis and visualizations.

## Conclusion
The analysis helps identify popular genres, highly voted movies, and patterns in Netflix movie releases.
